# HTTP 란

Hyper Text Transfer Protocol 이라는 뜻으로, 인터넷에서 데이터를 주고 받을 수 있게 해주는 프로토콜이다. 웹 브라우저가 서버에게 `request`를 보내면 서버는 브라우저에게 `response`한다. 이때 지키는 대표적인 통신규약이 `HTTP`이다. 크롬 개발자 도구를 통해 요청 헤더와 응답 헤더를 볼 수 있으며, 서버는 정보를 제공하면서 응답한 정보를 헤더를 통해 알려준다.
